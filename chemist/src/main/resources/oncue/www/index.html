Chemist Control Panel

Distribution Resources

GET /distribution: Display the current distribution of shards and associated work.

GET /lifecycle/history: View a rolling snapshot of the last lifecycle events to took place.

POST /distribute: Manually instruct a given set of inputs to be sharded over avalible Flask instances.

POST /bootstrap: Manually force Chemist to re-read its state of the world from AWS.

Shard Resources

GET /shards: List all shards and known information about those hosts

GET /shard/:shard-id: Displays all known information about a given shard.

POST /shard/:shard-id/include: Manually specify a machine to use as a load-bearing shard.

POST /shard/:shard-id/exclude: Manually remove a shard from rotation and reshard its work.
