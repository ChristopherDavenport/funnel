Chemist Control Panel

Control Resources

GET /distribution: Display the current distribution of shards and associated work.

GET /unmonitorable: Display a list of all the targets that were not monitorable.

GET /platform/history: Audit trail for the events that chemist recieved from the platform.

Shard Resources

GET /shards: List all shards and known information about those hosts

GET /shard/:id: Displays all known information about a given shard.

GET /shard/:id/sources: Reach out to the specified shard and request its workload.
